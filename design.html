Crazy Stack

Take a look at how we got from start to finish


Gameplay




Initial Concept

To decide what kind of web game to make we first researched other web games for ones which we enjoyed.
We found some stacking games and enjoyed how simple the idea is but also how fun it could be to play. We started brainstorming about different
ideas for levels and ways of making the game more interesting.
We decided that we would make a limit on how low you can drop the object to make it a little more
difficult to see where the shapes would drop. This would also mean that the objects would be hitting other objects with a bit more force than
if you drop it from just above the previous object.

Final Concept

While making the game we had many ideas of different ways to make the game harder and more fun.
One thing we tried was to add was a level which would choose a random shape we have made after every object has been dropped so that
you never know what the next object coming is. This was more difficult than we originally thought and we couldn't get it to work properly.
We were also going to try to have a level where you have to try stack as many objects as possible before they all fall over but to do this
we had to add vertical scrolling so that you are not limited by the height of the game window.
We were struggling to have the collisions and boundaries working with this so we decided to take it out.
